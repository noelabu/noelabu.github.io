Development of a Model for Smart Weed Control System with Herbicide Persistence Considerations utilizing Supervised Learning

Bunag, Noela Jean S., Elazegui, Diann Claire L., Natulan, James Bryan E.

Malayan Colleges Laguna

ABSTRACT

Soil pH influences the persistence of herbicides.Acidic soils allow a rapid dissipation of herbicides on soil while in alkaline soils herbicides persist much longer. As a response in this dilemma, a model that utilizes machine learning and machine vision was developed to distinguish weeds from crops and considers the soil pH for weed management. The weed detection system achieves an average detection accuracy of 90%, the soil pH determination system, that uses KNN algorithm with 64%, and the decision making system ,that uses Decision Tree algorithm, with 98%.

BACKGROUND and SIGNIFICANCE OF THE STUDY

One of the factors that affect the quality and quantity of crops is due to stress factors subjected to weeds. Weeds are unwanted plants that compete for nutrients, light, and moisture in a field of crop. Managing weeds by chemical means is one of the most common methods. However, the application of herbicides had also presented negative impacts such as soil persistence, which is the length of time an herbicide is active in the soil (Curran, 2016).  Existing devices like Precision Spray technologies had reportedly reduced the quantity of herbicide use that results in reduced costs, crop damage as well as reduces environmental and health risks. Nevertheless, these technologies had not incorporated the effects of herbicide on soil and the effect of the soil pH in the persistence and degradation of herbicides that makes this study a significant contribution to the recent trends in precision agriculture in terms of reducing the potential risks of herbicide use.

METHODOLOGY

Figure 1. Block Diagram of the System

Weed Detection System

For the weed detection, the model used was the Single Shot Detector (SSD) framework of Liu et al. (2016). The SSD approach is based on a feed-forwarding convolutional network that produces a fixed-size collection of bounding boxes and scores for the presence of object class instances on those boxes. The SSD model uses Mobilenet-V2 base network for its feature map extraction. The framework was trained using 250 images of targets and non-targets labeled manually for each target position on the images. It was trained containing the picture of weeds as the target object and the crop as non-targets.The network was run on NVIDIA Jetson Nano Developer Kit.

The output from the weed detection model is an image with the detected weeds inside the default boundary boxes. The number of weeds will be obtained by counting the objects detected whose detection scores are greater than 0.6. The location of the weeds was given by the center coordinates of the bounding box.

Soil pH Determination

The first step in the soil pH determination process is segmenting out the crop and weeds from the image to isolate the soil. This is done by removing the green parts in the image using  the image processing library of OpenCV. The imported raw image is first converted to the HSV color space for a better separation of colors. A mask is then created by declaring a range of values which will select the green areas in the image. The image and the mask would then be compared using a bitwise and operation which would result in a segmented image. The average value per channel (e.g. Red, Green, Blue) of the image would then be determined. The RGB values will be the input of a machine learning model which will predict the soil pH.

7 types of machine learning algorithms were trained on the created dataset of RGB values and corresponding soil pH. The K Nearest Neighbors algorithm performed best so the model was developed based on this algorithm as shown in Table 1. The output of this system is whether the soil is acidic, good, or alkaline.

Decision Making

The model used for decision making was based on the Decision Tree algorithm as it provides higher accuracy compared to other supervised learning algorithms based on table 1. The dataset consists of soil pH values, weed counts, and its respective class was divided into attributes and labels and into training and testing sets using Scikit-Learn library. The trained data was then fitted to the Decision Tree algorithm to determine the accuracy of the model and to predict classes based upon the two attributes of the image which are the soil pH value and the weed count. The class for the new data was predicted using the predict() function in the Scikit-Learn Library. The output for this system is whether to spray or not spray.

CONCLUSION

A robot that utilizes artificial intelligence for the application of weed management was developed. The robot includes the effects of herbicide use in soil and the effects of soil pH in the persistence of herbicides on soil. The robot is divided into three parts: the weed detection system, soil pH determination, and decision-making system.

  • The weed detection system distinguishes the weeds from crops and determines the number of weeds present in the image. The weed detection model yields an average detection accuracy of 90%.
  • The soil pH determination system isolates the soil in the image and determines whether the soil is acidic, good, or alkaline based on its color using the KNN algorithm with an accuracy of 64%.
  • The decision making system determines whether to spray or not depending on the soil pH and number of weeds present in the image using the Decision Tree algorithm with an accuracy of 98%.

For future works, additional data for the soil ph determination system will be gathered for a higher accuracy. The model will also be implemented in a smart sprayer robot. The smart sprayer will be evaluated in vegetable fields for weed management and compare its performance to a traditional weed sprayer.
